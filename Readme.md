# Project Overview
Intercity Pro is a sophisticated taxi management platform designed for long-distance, inter-city travel. It provides users with a seamless booking experience and gives administrators a powerful dashboard to manage a fleet of drivers.
The project focuses on high-performance data handling, featuring a Modular JavaScript Architecture that separates data (Database), logic (Admin CRUD), and utility functions (Theme Toggle). This ensures the application is scalable, maintainable, and provides a top-tier user experience in both light and dark environments.

# Key Features
-  Administrators can add, view, update, and remove driver profiles with real-time DOM updates.
-  Includes case-insensitive name searching, category filtering, and a "Top Rated" isolation filter for elite drivers.
-  A global Navbar toggle that saves the user's preference to localStorage, ensuring the theme remains consistent across all 5+ pages.
-  Interactive elements (like the Delete button) are conditionally rendered based on an Admin Permission Object.
-  Utilizes 12+ JavaScript String methods to clean, format, and validate data before it enters the database.
